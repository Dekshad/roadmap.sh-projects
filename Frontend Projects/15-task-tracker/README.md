# Task Tracker 📝

A clean, lightweight, and interactive To-Do List application built with vanilla JavaScript. This project focuses on efficient state management and an Object-Oriented Programming (OOP) approach.



## ✨ Features

- **Add Tasks:** Create new tasks instantly by pressing `Enter` or clicking the add button.
- **Task Management:** - Toggle task completion with a checkbox.
    - Completed tasks are automatically moved to the bottom of the list and styled with a strikethrough.
    - Delete tasks from the list using the trash icon.
- **Dynamic Rendering:** The UI stays in sync with the data model through a dedicated render function.
- **Responsive Design:** A modern, mobile-friendly interface built with Flexbox.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom styling using CSS variables (Custom Properties) and Flexbox.
- **JavaScript (ES6+):** - Functional/OOP approach using Constructor functions.
    - Array manipulation (`sort`, `unshift`, `splice`).
    - Event-driven logic.
